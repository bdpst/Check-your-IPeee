# Check-your-IPeee

Batch-утилита для Windows, которая показывает локальный IPv4-адрес компьютера и подсеть, в которой он находится.

Скрипт не привязан к диапазону `192.168.1.x`: он берет активные IPv4-адреса сетевых интерфейсов и вычисляет подсеть по `PrefixLength`.

## Что выводится

- подсеть в формате `network/prefix`, например `192.168.10.0/24`;
- локальный IP-адрес;
- маска подсети;
- имя сетевого интерфейса;
- шлюз, если он есть.
- локальные DNS-серверы выбранного интерфейса.
- внешний IPv4-адрес и код страны, если доступен интернет.

Если на компьютере несколько активных интерфейсов, скрипт выводит основной адрес: интерфейс со шлюзом по умолчанию. Если шлюз не найден, выводится первый подходящий активный IPv4-адрес.

Блок внешнего IP необязательный. Если интернет недоступен или внешний сервис не отвечает, скрипт не ломается и показывает сообщение `Внешний IP: не удалось определить`.

## Запуск

Откройте:

```bat
Check-your-IPeee.bat
```

Требования: Windows 10/11 или Windows Server с PowerShell и сетевыми cmdlet `Get-NetIPConfiguration`, `Get-NetIPInterface`.
